# Ahooooooo

I'm building open-source tools that make AI-assisted software development easier
to verify and safer for human maintainers.

## Featured project: ReviewReady

**Know what is missing before a pull request consumes human review time.**

[ReviewReady](https://github.com/ahoooooooo/reviewready) turns a repository's own
contribution policy into deterministic evidence checks. It can require PR
documentation, linked issues, CI results, maintainer review, and explicit human
responsibility—without using an LLM to approve code.

## What I care about

- deterministic evidence instead of model judgment;
- repository-owned policy and secure defaults;
- human maintainers retaining review and merge authority;
- learning from real open-source maintainer workflows.
